Analyzing Weather by Latitude

Summary

The purpose of this project is to anaylze weather data from cities all over the world to determine different trends based on latitude. The four trends being looked at are temperature, humidity percentage, cloud percentage and wind speeds. To do this I used Python libraries and OpenWeather, to generate a random list of cities and find current weather data for cities.

The image above is a global heat map that shows where all the cities that were queried through OpenWeather. Using the information gathered, (a complete table of this data can be found on the "Data " page) I generated a series of graphs to display how weather conditions change depending on distance from the equater.

While this website mostly analyzes how temperature, humidity, clouds, and wind speeds vary throughout the world, correlation graphs of just the northern and southern hemispheres are available as well. Simply select "Plots " and select "More Plots". Enjoy!
